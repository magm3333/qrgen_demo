# QRGen Demo

Generador de códigos QR estático desarrollado con Vue 3 + Vite.

## Características

- Generación de QR en tiempo real
- Descarga en formatos PNG y JPG
- Editor de plantillas con imágenes
- Soporte para temas oscuro y claro
- Despliegue automático en GitHub Pages

## Tecnologías

- Vue 3
- Vite
- QRCodeStyling

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
